3.1.6.3 Machine Learning

1. Introducción

En este apartado, se describe la sección Analytics.

2. Escenario

Para trabajar con muchos datos, se ha procedido a la carga del ejemplo 'Sample web logs'.

3. Machine Learning

En el panel lateral izquierdo, en la zona de Analytics se encuentra la opción Machine Learning.

En esta sección se pueden analizar datos de los logs con la tecnología machine learning. La interfaz es la siguiente.

Las opciones son las siguientes:

  • Visualize data from a file: Selecciona un archivo JSON o CSV desde el que importar datos a analizar.
  • Visualize data from a data view: Selecciona los datos de alguno de los índices ya existentes.
  • Start trial: Permite la experiencia de pago de machine learning.

Para este caso, se van importar datos de los índices del servidor, es decir, Visualize data from a data view. Al pulsar el botón, aparece la página siguiente:

Como se puede observar en la siguiente figura, se puede seleccionar desde búsquedas guardadas o desde índices. Por defecto viene en búsquedas guardadas, así que como para este caso de ejemplo se va a seleccionar un índice entero, hay que pulsar el botón Types y cambiarlo a Index pattern. A continuación, hay que seleccionar el índice que interese analizar, para este caso de ejemplo, se selecciona kibana_sample_data_logs.

Una vez seleccionado, aparece la siguiente página:

En esta página, se pueden ver todos los datos que contenía el índice de forma aislada para que se pueda analizar. Por ejemplo, se puede ver que en el campo agent.keyword se encuentran 89 apariciones de mozilla Linux, 85 de Chrome y 64 del navegador de Windows.
