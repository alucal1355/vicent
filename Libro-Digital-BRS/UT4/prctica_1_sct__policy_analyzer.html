Práctica 1. SCT - Policy Analyzer

 

1. Introducción

En esta práctica aprenderás a utilizar la herramienta Policy Analizer para comprobar qué van a aportar las líneas base de configuración de seguridad de Microsoft a tu sistema (subdominio de tu proyecto).

2. Trabajo a realizar

2.1 Análisis

Analiza la línea base con la configuración actual de tu sistema (subdominio de tu proyecto) describiendo gráfica y textualmente los resultados obtenidos, es decir:

  • Directivas configuradas con distintos valores.
  • Directivas con valores en conflicto.
  • Nuevo conjunto de reglas que se han creado con la configuración efectiva.
  • etcétera.

Adjuntar al zip de entrega:

  • Fichero generado y nombrado de esta forma: MSFT Windows Server 2019 - Domain Controller (tu_nombre_apellidos).

2.2 Compara dos línea base

En  este apartado has de comparar las siguientes líneas base:

  • MSFT Windows Server 2019 - Domain Controller.
  • MSFT Windows Server 2019 - Member Server.

Describe gráfica y textualmente las diferencias.

3. Entregar

Entrega un fichero zip que contenga:

  • Documento en formato pdf con lo que se demanda en la práctica.
  • Fichero generado y nombrado de esta forma: MSFT Windows Server 2019 - Domain Controller (tu_nombre_apellidos) en el apartado 2.1. Análisis.
