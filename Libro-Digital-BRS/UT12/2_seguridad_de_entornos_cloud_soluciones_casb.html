2. Seguridad de entornos cloud. Soluciones CASB

1. ¿Qué es CASB?

CASB (Cloud Access Security Broker o Agente de Seguridad para el Acceso a la nube) es una herramienta intermediaria entre los datos y aplicaciones albergadas en la nube y el usuario con el fin de proteger el acceso a los recursos que se sitúan en la nube.

2. Características

Las principales características de los CASB son:

  • Visibilidad. Para proteger a los datos y a los usuarios las soluciones CASB proporcionan una serie de datos cómo pueden ser: estadísticas sobre el uso de las aplicaciones en la nube, información del usuario, su dispositivo y la ubicación en la que está trabajando.
  • Legalidad. Las empresas son responsables de cumplir con las normas sobre privacidad y seguridad de los datos. Para ello un CASB puede determinar las áreas que tienen mayor riesgo frente amenazas y ayudar a la empresa para solventar dichos problemas.
  • Seguridad. Las soluciones CASB implementan funciones de seguridad cómo pueden ser: perfiles de acceso, logs, tokenización, cifrado de conexiones, asignación de credenciales...
  • Protección frente amenazas. Pueden detectar y bloquear amenazas tan pronto como alguien intente obtener datos a los que no está autorizado u obtener acceso ilegítimo.

3. Funcionamiento

Aunque cada solución es diferente, todas ellas tienen una misma estructura:

  1. Recopila una lista de todos los servicios albergados en la nube junto con los usuarios que los utilizan.
  2. Una vez recopilada toda la lista, el CASB evalúa el nivel de riesgo que tiene cada aplicación. Para evaluar dicho nivel tiene en cuenta qué aplicación es, qué tipos de datos tiene y cómo se acceden a dichos datos.
  3. Teniendo ya el riesgo de cada aplicación, el CASB evalúa dichos riesgos para establecer una política de acceso a los usuarios para las aplicaciones y tomar las medidas oportunas cuando se produce una infracción.
